Projects

This To-Do List application is a dynamic tool for efficient task management, featuring task creation, categorization into 'Business' and 'Personal', and persistent local storage. Designed with a responsive layout for compatibility across devices, it leverages HTML5, CSS3, and JavaScript for a seamless user experience. Future updates will introduce new categories, structured task display, and deadline setting functionalities.

This Java implementation of the classic Snake game serves as a practical exercise in GUI development, event handling, and game logic using Java Swing. It features engaging gameplay with intuitive keyboard controls, dynamic collision detection, real-time score tracking, and offers a comparison between switch/case and if/else logic. This project is an excellent resource for understanding color handling in Java and exploring GUI development nuances, making it a valuable learning tool for Java enthusiasts.
